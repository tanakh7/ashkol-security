🔧 הגדרת Firebase (חובה)

כדי שהעדכונים יגיעו לכל המאבטחים בזמן אמת, יש להגדיר Firebase פעם אחת.

הוראות הגדרה (5 דקות):

  1. כנס ל־console.firebase.google.com
  2. לחץ "Add project" ← שם כלשהו ← Continue
  3. בתפריט הצדדי: Build → Realtime Database → Create database
  4. בחר "Start in test mode" ← Enable
  5. לחץ על ⚙️ ← Project settings ← גלול ל־"Your apps" ← לחץ </>
  6. רשום שם אפליקציה ← Register app
  7. העתק את ה־firebaseConfig והדבק בקובץ ה־HTML (בתחילת הקובץ, ב־initializeApp)
  8. העלה מחדש ל-GitHub — האתר יתחבר אוטומטית
